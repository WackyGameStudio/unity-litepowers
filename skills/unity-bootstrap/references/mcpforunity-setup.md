# MCPForUnity Setup

## Manual Install Path

1. In Unity, open Package Manager.
2. Add package from Git URL:

```text
https://github.com/CoplayDev/unity-mcp.git?path=/MCPForUnity#main
```

3. Open `Window > MCP for Unity`.
4. Run setup wizard or client configurator.
5. Restart or refresh AI coding agent if tools are not visible.

## Required Evidence

- MCPForUnity package is installed or explicitly missing.
- Active Unity project path is known.
- `Application.dataPath` matches target project.
- Required tool groups are available for planned work.
- Console/import status is fresh.

If MCPForUnity is unavailable, record evidence gap and avoid Editor-backed completion claims.
