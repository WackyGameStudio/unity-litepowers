# MCPForUnity UI Scene Playbook

## Scene

- Confirm target identity.
- Load scene.
- Inspect hierarchy.
- Apply planned changes.
- Save scene only when requested by plan.
- Read console after import/refresh.

## Prefab

- Inspect prefab path.
- Verify required components.
- Verify serialized references.
- Run prefab smoke or PlayMode evidence when behavior depends on wiring.

## UI Toolkit

- Inspect UIDocument/UXML/USS.
- Use UI evidence for layout and callback when available.

## uGUI

- Inspect Canvas hierarchy, components, TextMesh Pro, Button callbacks, navigation, and prefab wiring.
- Use screenshot or PlayMode smoke for visual/callback claims.
